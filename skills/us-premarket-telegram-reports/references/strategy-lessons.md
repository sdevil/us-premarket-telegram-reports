# Strategy lessons

Use this file to accumulate durable lessons from post-market review.

## Update rules

When a review finds a meaningful mismatch between forecast and actual outcome:

1. Identify the reason category:
   - macro regime mismatch
   - sector mismatch
   - ticker-specific catalyst mismatch
   - structure / volume mismatch
   - execution-rule mismatch
2. Convert the lesson into a short reusable rule.
3. Prefer rules that can be applied again.
4. Avoid vague statements like "be more careful".

## Format

### YYYY-MM-DD
- Ticker / theme:
- Problem type:
- What was missed:
- Reusable lesson:
- Future watchpoint:

## Example style

### Example
- Ticker / theme: TSLA
- Problem type: ticker-specific catalyst mismatch
- What was missed: stock was treated like a generic AI-beta name without a fresh Musk/Tesla-specific driver
- Reusable lesson: do not rank TSLA as high-conviction unless there is either a true company catalyst or exceptional opening-drive confirmation
- Future watchpoint: check Musk/Tesla posts and delivery/FSD context before upgrading TSLA
